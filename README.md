# codeToSlide
Sistema que gera slide para aula com trecho de códigos e explicações baseadas em comentários.

# Funcionalidades
* Importar código
* Gerar slide html
* Gerar slide PDF
* Criar modelos
* Configuração de estilo (cor de background, cor de fonte, tipo de fonte, tamanho da fonte)
